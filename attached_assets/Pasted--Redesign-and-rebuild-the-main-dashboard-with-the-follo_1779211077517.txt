"Redesign and rebuild the main dashboard with the following changes all applied together.
1. Merge Weekly Review into Dashboard
Remove the Weekly Review page from the sidebar navigation entirely. Merge all useful content from it into the main dashboard as a collapsible Weekly Summary section at the bottom. The Weekly Summary should be collapsed by default and expand to show: total tasks completed that week, goals completed that week, a mood trend line, and a tasks completed per day bar chart.
2. Remove the Following Elements
Remove the Weekly Score card. Remove the Habit Streaks section and all habit-related charts and cards from the dashboard and only show them when the user has set up at least one habit. Remove the Focus Time card and Focus minutes per day chart and only show them when the user has used the focus timer at least once. Remove all empty charts and cards that show zero data with no user activity behind them.
3. Unified Calendar Date Filter
Add a single unified calendar picker at the top of the dashboard that opens in a floating panel when clicked. Clicking a single date selects just that day. Clicking and dragging across multiple dates selects a custom range and highlights all selected dates. Clicking the month name at the top of the calendar selects the entire month. Clicking the year selects the entire year. The left side of the picker should show the following predefined shortcuts: Today, Yesterday, This Week, Last Week, This Month, Last 30 Days, This Quarter, and This Year. Display the selected period in a clean pill-shaped button at the top of the dashboard showing the selection in plain language such as Today, May 13 — May 19, or May 2026. All dashboard sections should update in real time as the date selection changes. The calendar should close when the user clicks outside it or selects a shortcut. On mobile the picker should open as a bottom sheet instead of a floating panel.
4. Overdue Alert Banner
Add an overdue alert banner directly below the date filter that only appears when there are overdue tasks or goals. It should show a count of overdue tasks and overdue goals separately with a clickable link that navigates to the filtered view of each. The banner should be red or amber and dismissible for the session but reappear the next day if items are still overdue.
5. Stat Cards Row
Keep a row of four stat cards showing: tasks completed versus total due for the selected period, number of active goals, today's mood if one has been selected, and productivity score. Remove any stat card that shows only zero with no activity behind it.
6. Three Panel Main Layout
Below the stat cards show a three panel layout as follows.
Left panel — To-Do List: Show all tasks due in the selected date period. Sort them with overdue tasks at the top highlighted in red, then tasks due today, then upcoming tasks. Each task should have a checkbox to mark it complete directly from the dashboard without navigating away. Completed tasks should move to the bottom of the list with a strikethrough. Show the priority badge and due date on each task card.
Centre panel — Active Goals: Show active goals with a checkbox to mark them complete directly from the dashboard. Show the due date, priority badge, and a dropdown on each goal card to update the status to Not Started, In Progress, Completed, Paused, or Archived without navigating away. When a goal is marked complete move it to the bottom of the panel with a strikethrough and update the Completed tab on the Goals page automatically.
Right panel — Upcoming 7 Days: Show all tasks and goals due in the next 7 days regardless of the date filter, grouped by date. Each item should be clickable and open the full detail view.
